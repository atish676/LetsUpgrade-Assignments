There are 4 methods to access elements in JavaScript:

ID
document.getElementById is the method of accessing elements using the ID attribute by giving them a special ID with which the elements can be identified. This method returns the element with that ID. For eg-

Class
document.getElementsByClassName is the method of accessing elements using the class attribute by giving them a special class name with which the elements can be identified. It can also be used to access elements with predefined class names. This method returns a collection, so to access the elements we need to specify the index or use a loop. For eg-

Tag Name
document.getElementsByTagName is the method of accessing elements using the name of tag of elements we want to access. This also returns a collection, so to access the elements we need to specify the index or use a loop. For eg-

CSS Query Selector
document.querySelector is the method of accessing elements using any of the above methods. Use . for classes, # for Ids and name of tag for the tag names. It returns the first element that encounters the match. document.querySelectorAll is another method in which all the elements are returned called NodeList and to access the elements we need to specify the index or use a loop.
